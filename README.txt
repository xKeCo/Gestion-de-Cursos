**INSTRUCCIÓNES (Base de Datos)**

Para importar la BD en mySQL en PHPmyAdmin es necesario crear un base de datos con el mismo nombre del archivo .SQL que esta alojado en la carpeta principal del proyecto. posteriormente importe el archivo SQL para asi poder ejecutar exitosamente el proyecto. Gracias

**INSTRUCCIÓNES (Visualizacion del proyecto)**

Debe cloner el repositorio en la carpeta C:\xampp\htdocs, abrir xampp y correr Apache y mySQL, porteriormente ingresar al navegador e ir a la pagina http://localhost/ProyectoFInalEDyA2/ y listo.
